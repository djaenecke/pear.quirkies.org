The Quirkies.org PEAR Channel

Registering the channel: 
pear channel-discover pear.quirkies.org
Listing available packages: 
pear remote-list -c quirkies
Installing a package: 
pear install quirkies/package_name
Installing a specific version/stability: 
pear install quirkies/package_name-1.0.0
pear install quirkies/package_name-beta
Receiving updates via a feed: 
http://pear.quirkies.org/feed.xml

Psr_Log - Port of the psr/log composer package

The Psr_Log package provides the reference implementation of the logging facilities defined in PSR-3. It is a port of the original psr/log Composer package, enabling authors to declare the depenency in a package.xml file

Install commandpear install quirkies/Psr_Log
